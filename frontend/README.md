# Flutter Frontend (`frontend/`)

This directory will contain the Flutter codebase for the desktop user interface.

## Key Features

- **Responsive Design**: Premium dark mode theme with glassmorphic visuals and fluid transitions.
- **Batch Processing**: Semicolon-separated path handler to score multiple files sequentially.
- **Dynamic Configuration**: Interactive channel selections (EEG, reference, EOG, EMG) scanned directly from files.
- **Real-Time Monitor**: Log console displaying subprocess execution stdout and progress bar indicators.
